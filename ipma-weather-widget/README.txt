== Description ==

The Weather Forecast in Portugal, provided by the IPMA plugin, lets you display a real-time forecast for any location in Portugal (mainland and archipelagos) in a WordPress website. This plugin displays the following data: icon indicating current weather conditions, high and low temperatures measured in Celsius, and chance of precipitation for today, tomorrow, and the day after.

This plugin provides a weather widget you can place on your website’s sidebar, or add it to a page or theme using the following shortcode:

	[ipma-wiget]

=== Simple ===

The use of this plugin requires no knowledge in programming and no code writing is necessary. Simply install it, place the widget in a sidebar or insert it via shortcode, and the plugin will work correctly.

=== Mobile responsive ===

This plugin is optimized to work properly on mobile devices.

== Installation ==


1. Upload `ipma-widget` to the WordPress `plugins` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Insert the `Weather Forecast in Portugal` widget in your sidebar

== Frequently Asked Questions ==

= Can I use the widget outside a sidebar? =

Yes, simply use the shortcode provided above that can have the following attributes:

* title: Allow the use of an alternative title above the selects
* horizontal: Displays the widget on a full-width responsive mode

= This plugin uses cookies to store the users information? =

No, the options selected by the user are reloaded on the next visited, being such options stored in the browser localStorage.
